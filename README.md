## Project Overview
This Power BI project analyzes company sales data to identify trends, top-performing products, regional performance, and key business insights. The dashboard is designed to support quick and effective decision-making for business growth.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- SQL
- Excel

## Dataset
- 10,000+ sales records  
- Key columns: Date, Customer, Product, Region, Quantity, Sales, Profit  
- Data cleaned and transformed using Power Query

## Key Features
- KPI Cards: Total Sales, Total Profit, Average Order Value
- Monthly & Yearly Sales Trend Analysis  
- Region-wise and Product-wise performance visuals  
- Top/Bottom N products and customers  
- Drill-through pages  
- Interactive slicers & filters  
- Custom DAX Measures (YoY Growth, % Contribution, Running Totals)

## Steps Performed
1. Imported dataset (Excel/CSV/SQL)
2. Cleaned data: removed nulls, corrected types
3. Created Calendar Table & Star Schema Model
4. Built DAX measures for KPIs and calculations
5. Designed dashboard visuals and layout
6. Added drill-through and interactive filters
7. Optimized Refresh and Data Model

## Key Insights
- 15% Year-over-Year sales growth
- Top 5 products generate ~45% of total revenue
- South region is the top-performing region
- Returning customers contribute higher revenue than new customers

## How to Use
1. Download the `.pbix` file and dataset.
2. Open the PBIX file in Power BI Desktop.
3. Refresh data if needed.
4. Use slicers and buttons to explore insights.
